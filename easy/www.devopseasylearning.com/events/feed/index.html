Event Description

This program is completely hands-on oriented and designed to help students become Linux Systems Administrator, DevOps Engineer and AWS Admin. It will provide you with an in-depth knowledge of Linux/Unix concepts, Amazon web services (AWS), software development life cycle (SDLC), Infrastructure as Code (IaC) with Terraform and Terragrunt, bash scripting, and various DevOps tools including Git, Git-crypt, Bitbucket, Jenkins, Maven, Docker, Helm, Ansible, Kubernetes or K8s, Node, Golang, Sonarqube, Sonar-scanner, Nexus, docker-compose, GKE, RKE, Rancher, ArgoCD, Spinnaker, MetalLB, Pgadmin, Postgres, HA-proxy, Nginx reverse proxy-manager, Redis-Cluster, Apache Kafka and Zookeeper, ElasticSearch-Cluster, etc.

Topic to be discuss:

  • Introduction to Computer and Networking
  • Linux Bash Shell Scripting
  • Linux System Administrator
  • Amazon Web Services (AWS)
  • Agile Methodology
  • Manage Workflow with Jira
  • Overview Of DevOps
  • Version Control With Git, Bitbucket and Github
  • Continuous Integration Using Jenkins
  • Build Automation and Code Analysis with Maven, Node, Golang, Sonarqube, Sonar-scanner
  • Configuration Management with Ansible
  • Containerization With Docker
  • Container Orchestration, Deployment strategy and Backup with Kubernetes, AWS EKS, GKE, RKE, Rancher, ArgoCD, Spinnaker, MetalLB
  • Application Deployment In Kubernetes with Helm
  • Infrastructure Monitoring with ELK, filebeat, Metricbeat, fluentd
  • Infrastructure as Code With Terraform
  • Terraform Management With Terragrunt
  • Distributed Systems: Redis-Cluster, Apache Kafka and Zookeeper, ElasticSearch-Cluster
  • Proxy and LoadBalancer: HA-proxy, Nginx reverse proxy-manager
  • Databases : MySQL, Redis, Pgadmin, Postgres
